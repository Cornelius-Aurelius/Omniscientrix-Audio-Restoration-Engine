# Omniscientrix Audio Restoration Engine  
Ultra-clean AI audio enhancement using the Omniscientrix Intelligence Engine.

## Features
- Noise removal  
- Echo reduction  
- Spectral smoothing  
- Hyperdiffusion clarity boost  
- Clean WAV export  
- Works in Google Colab, Python, and web interfaces

## How to Use (Colab)

1. Open the `.ipynb` or paste the Python file into a new Google Colab.
2. Upload any audio file (mp3, wav, m4a, ogg).
3. The engine automatically:
   - Converts the file  
   - Restores audio  
   - Outputs a cleaned WAV file
